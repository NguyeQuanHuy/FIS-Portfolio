## 🚀 Highlight-Projekte

### 🌐 01 — Heimnetzwerk mit VLAN-Trennung
> **Schwierigkeit:** ⭐⭐⭐ Mittel · **Tools:** Cisco Packet Tracer

Aufbau eines simulierten Firmennetzwerks mit drei VLANs (Office, Guest, Server), Inter-VLAN-Routing über Router-on-a-Stick und ACL-basierter Trennung.

📁 [→ Zur vollständigen Doku](./projects/01-vlan-network/)

---

### 🪟 02 — Windows Server + Active Directory
> **Schwierigkeit:** ⭐⭐⭐⭐ Mittel-Fortgeschritten · **Tools:** Windows Server 2022, VMware

Komplette Domänenumgebung mit OUs, Benutzerverwaltung, Gruppenrichtlinien (GPO), DNS, DHCP und Dateifreigaben mit NTFS-Berechtigungen.

📁 [→ Zur vollständigen Doku](./projects/02-active-directory/)

---

### 🐧 03 — Linux-Webserver mit SSH-Härtung
> **Schwierigkeit:** ⭐⭐⭐ Mittel · **Tools:** Ubuntu 22.04, Apache2, UFW

Ubuntu-Server als Webserver mit Key-basierter SSH-Authentifizierung, deaktiviertem Root-Login, UFW-Firewall und automatisiertem Backup via Cron.

📁 [→ Zur vollständigen Doku](./projects/03-linux-webserver/)

---

### 💾 04 — Backup- & Wiederherstellungssystem
> **Schwierigkeit:** ⭐⭐⭐ Mittel · **Tools:** Windows Server Backup, rsync

Implementierung der 3-2-1-Backup-Regel: 3 Kopien, 2 Medien, 1 extern. Inkl. Restore-Test und RTO/RPO-Dokumentation.

📁 [→ Zur vollständigen Doku](./projects/04-backup-system/)

---

### ⚙️ 05 — PowerShell: AD-Benutzer aus CSV
> **Schwierigkeit:** ⭐⭐⭐⭐ Fortgeschritten · **Tools:** PowerShell, AD-Module

Automatisierte Massenanlage von AD-Benutzern aus einer CSV-Datei mit Logging, Fehlerbehandlung und OU-Zuordnung pro Abteilung.

📁 [→ Zur vollständigen Doku](./projects/05-ad-automation/)

---

### 🏗️ 06 — Virtuelles Homelab
> **Schwierigkeit:** ⭐⭐⭐ Mittel · **Tools:** VMware, pfSense

Dauerhafte Mehr-VM-Umgebung mit pfSense-Firewall, Windows-Domäne und Linux-Server — zum kontinuierlichen Lernen und Testen.

📁 [→ Zur vollständigen Doku](./projects/06-homelab/)
